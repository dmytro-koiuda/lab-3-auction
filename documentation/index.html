Документація до програми "Аукціон"

Ця програма реалізує аукціонний процес з продажу товарів учасникам за допомогою мультипотокового програмування.

Класи

Auction

Цей клас відповідає за ініціалізацію та керування аукціоном.

Методи:

  • startAuction(): Розпочинає аукціон та виводить інформацію про його стан після завершення.
  • conductAuction(Item item): Здійснює аукціон для конкретного товару.
  • sellItem(Item item): Продає товар, знаходить переможця та зменшує баланс його акаунта.
  • findWinner(Item item): Знаходить переможця аукціону для конкретного товару.
  • main(String[] args): Точка входу в програму.

Item

Представляє товар, який продається на аукціоні.

Поля:

  • name: Назва товару.
  • startPrice: Початкова ціна товару.
  • currentPrice: Поточна ціна товару.
  • owner: Власник товару.

Методи:

  • setCurrentPrice(double currentPrice): Встановлює поточну ціну товару.
  • setOwner(Participant owner): Встановлює власника товару.

Participant

Представляє учасника аукціону.

Поля:

  • balance: Баланс учасника.
  • name: Ім'я учасника.

Bid

Представляє ставку на аукціоні.

Поля:

  • amount: Сума ставки.
  • participant: Учасник, що зробив ставку.
  • item: Товар, на який зроблено ставку.

Використані технології

  • Java: Мова програмування, у якій реалізовано програму.
  • Java Concurrency API: Використовується для роботи з потоками та виконання мультипотокового коду.

Параметри аукціону

  • RISK_COEFFICIENT: Коефіцієнт ризику для визначення максимальної ставки.
  • ITEM_SOLD_TIME: Час, протягом якого триває аукціон для кожного товару.

Завершення аукціону

Після завершення аукціону виводиться інформація про продані товари, включаючи їх назви та ціни.

Використання

Для запуску програми виконайте метод main(String[] args) класу Auction.
